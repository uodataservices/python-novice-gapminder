This lesson is an introduction to programming in Python 3 for people with little or no previous
programming experience.

This lesson teaches Python using [JupyterLab](https://jupyterlab.readthedocs.io/en/latest/).

::::::::::::::::::::::::::::::::::::::::::  prereq

## Prerequisites

1. Learners must come with a laptop on which they have privileges sufficient to install Python.
Phones and tablets are not acceptable.
2. While alternative JupyterLab installations are acceptable, the first class session is dedicated to the installation of Python and JupyterLab.
